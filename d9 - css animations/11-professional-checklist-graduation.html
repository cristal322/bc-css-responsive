✅ Checklist del Senior CSS Developer

⚡
Performance Optimization

  • Usar solo propiedades GPU-friendly en animaciones
    transform, opacity, filter - nunca width, height, top, left
  • Implementar will-change strategy correctamente
    Preparar capas GPU antes de animaciones, remover después
  • Optimizar critical rendering path
    Critical CSS inline, defer non-critical styles
  • Monitoring de performance en tiempo real
    FPS tracking, frame time analysis, reflow detection

📱
Cross-Device Compatibility

  • Testing en dispositivos de gama baja
    Validar performance en hardware limitado (2-4 cores, 2-4GB RAM)
  • Adaptive performance basado en device capabilities
    navigator.hardwareConcurrency, navigator.deviceMemory
  • Network-aware optimizations
    Ajustar assets y animaciones según connection speed
  • Cross-browser compatibility testing
    Safari, Chrome, Firefox, Edge - mobile y desktop

♿
Accessibility Standards

  • Respect prefers-reduced-motion
    Mandatory compliance - reduce/eliminate animations for sensitive users
  • Keyboard navigation support
    Focus states, tab order, keyboard shortcuts
  • Screen reader compatibility
    Semantic HTML, ARIA labels, meaningful focus management
  • Color contrast and visual accessibility
    WCAG AA compliance, color-blind friendly palettes

🔧
Code Quality Standards

  • Design tokens system implementation
    CSS custom properties, consistent spacing/color/typography scales
  • Maintainable architecture patterns
    Component-based CSS, clear naming conventions, documentation
  • Performance budgets and monitoring
    Bundle size limits, performance regression detection
  • Production-ready optimization pipeline
    CSS minification, tree-shaking, critical CSS extraction
