Configuración CSS-First en Acción

Observa cómo la nueva sintaxis @theme hace el desarrollo más intuitivo 

/* Nueva configuración TailwindCSS v4.0 */
@theme {
  --color-brand-500: #0ea5e9;
  --color-brand-600: #0284c7;
  --font-display: 'Poppins', sans-serif;
  
  /* Variables disponibles en runtime */
  --animate-float: float 3s ease-in-out infinite;
}

@layer components {
  .btn-primary {
    @apply bg-brand-500 hover:bg-brand-600 
           text-white px-6 py-3 rounded-lg 
           font-semibold transition-colors;
  }
}
