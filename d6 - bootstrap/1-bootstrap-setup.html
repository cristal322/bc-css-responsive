Nuevas Características Bootstrap 5.3+

Modo Oscuro Nativo

Implementación nativa del modo oscuro sin JavaScript adicional. Utiliza el atributo data-bs-theme para cambios instantáneos.

Variables CSS Modernas

Personalización en tiempo real usando CSS Custom Properties. Cambia colores y espaciado sin recompilar.

Breakpoint XXL

Nuevo breakpoint para pantallas ultra-wide (1400px+). Perfecto para monitores modernos y displays 4K.
