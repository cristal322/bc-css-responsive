Auditoría de Accesibilidad WCAG 2.1

  • ✓
    Contraste de Colores
    Ratio de contraste mínimo 4.5:1 para texto normal y 3:1 para texto grande.
    color: #334155; /* Contraste 7.2:1 sobre fondo blanco */
  • ✓
    Navegación con Teclado
    Todos los elementos interactivos son accesibles via teclado con orden lógico de tabulación.
    *:focus { outline: 2px solid #2563eb; outline-offset: 2px; }
  • ✓
    Texto Alternativo en Imágenes
    Todas las imágenes informativas tienen descripciones alt apropiadas.
    <img alt="Gráfico mostrando aumento del 25% en ventas Q4 2024">
  • ✓
    Estructura Semántica HTML
    Uso correcto de elementos semánticos (header, nav, main, section, article).
    <main><article><h2>Título</h2><p>Contenido...</p></article></main>
  • ✓
    Etiquetas ARIA
    Uso apropiado de atributos ARIA para mejorar la experiencia con lectores de pantalla.
    <button aria-expanded="false" aria-controls="menu">Menú</button>
  • ⚠
    Animaciones y Movimiento
    Respeta prefers-reduced-motion, pero algunas animaciones podrían optimizarse más.
    @media (prefers-reduced-motion: reduce) { * { animation: none; } }
  • ✓
    Escalabilidad de Texto
    El contenido es legible y funcional hasta 200% de zoom sin scroll horizontal.
    font-size: clamp(1rem, 2.5vw, 1.125rem);

Herramientas de Auditoría Recomendadas

Lighthouse (Chrome DevTools)
Performance • Accesibilidad • SEO
Auditoría integrada en Chrome que analiza performance, accesibilidad, mejores prácticas y SEO. Fundamental para cualquier proyecto web.
axe DevTools
Accesibilidad
Extensión de navegador que encuentra automáticamente problemas de accesibilidad con sugerencias específicas de solución.
WAVE (Web Accessibility Evaluator)
Accesibilidad
Herramienta visual que muestra directamente en la página los elementos de accesibilidad y errores detectados.
WebPageTest
Performance
Análisis detallado de performance desde múltiples ubicaciones y dispositivos reales, con waterfall charts y métricas avanzadas.
Color Contrast Analyser
Accesibilidad
Herramienta para verificar ratios de contraste entre colores de texto y fondo según estándares WCAG.
Responsive Design Checker
Responsive Design
Testing cross-device para verificar que el diseño responsive funciona correctamente en diferentes resoluciones.
