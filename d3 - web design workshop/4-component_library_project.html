Propósito: El Hero Section es el primer impacto visual de tu página. Utiliza Flexbox para centrar perfectamente el contenido y crear una distribución armoniosa de elementos. Este componente demuestra el poder de flex-direction: column combinado con centrado perfecto.

🔧 Código del Hero Section:
/* Hero con Flexbox - Centrado Perfecto */ .hero-content { display: flex; flex-direction: column; /* Stack vertical */ align-items: center; /* Centrado horizontal */ gap: 25px; /* Espaciado consistente */ max-width: 800px; margin: 0 auto; } .hero-actions { display: flex; /* Botones lado a lado */ gap: 20px; /* Espacio entre botones */ flex-wrap: wrap; /* Responsive wrapping */ justify-content: center; /* Centrado horizontal */ }
🎯 Concepto Clave - Hero Responsive:

El Hero Section combina múltiples niveles de Flexbox. El contenedor principal usa flex-direction: column para apilar elementos verticalmente, mientras que hero-actions usa dirección horizontal para los botones. Esta técnica de "nested flexbox" es fundamental en el diseño moderno.

🔧 Código de Feature Cards:
/* Grid de Cards con Flexbox Responsive */ .features-grid { display: flex; flex-wrap: wrap; /* Permite salto de línea */ gap: 30px; /* Espaciado entre cards */ } .feature-card { flex: 1 1 300px; /* Grow, shrink, base de 300px */ display: flex; flex-direction: column; /* Contenido vertical */ gap: 20px; /* Espaciado interno */ } .feature-header { display: flex; /* Ícono + título horizontal */ align-items: center; /* Centrado vertical */ gap: 15px; /* Espacio ícono-texto */ }
🎯 Concepto Clave - Cards Inteligentes:

La magia está en flex: 1 1 300px. Esto significa: "crece para llenar el espacio (1), encógete si es necesario (1), pero nunca bajes de 300px de ancho". Esta técnica crea layouts que se ven profesionales en cualquier pantalla sin media queries complejas.

🔧 Código de Pricing Cards:
/* Pricing Cards con Altura Uniforme */ .pricing-card { display: flex; flex-direction: column; /* Stack vertical */ gap: 25px; /* Espaciado consistente */ } .pricing-features { display: flex; flex-direction: column; /* Lista vertical */ gap: 15px; /* Espacio entre features */ flex-grow: 1; /* Toma todo el espacio disponible */ } .pricing-feature { display: flex; /* Check + texto horizontal */ align-items: center; /* Centrado vertical */ gap: 10px; /* Espacio check-texto */ }
🎯 Concepto Clave - Altura Uniforme:

La clave del éxito en pricing cards está en flex-grow: 1 aplicado a la lista de features. Esto hace que las listas se expandan para llenar el espacio disponible, manteniendo todos los botones de acción alineados en la parte inferior, sin importar cuánto contenido tenga cada card.
